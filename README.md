# Brain Tumor Detection using CNN

VGG16-based deep learning model for classifying brain MRI images into:
- Glioma
- Meningioma
- Pituitary
- No Tumor

## Model
The trained model is hosted on Google Drive and is automatically downloaded
when running `app.py` or `predict.py`.

## Run Streamlit App
```bash
pip install -r requirements.txt
streamlit run app.py
